# Puzzle Rick and Morty

¡Bienvenido al proyecto **Puzzle Rick and Morty**! Este es un juego interactivo basado en los personajes de la famosa serie "Rick and Morty". El objetivo es completar puzzles de los personajes mientras exploras el multiverso.

## Características

- Selección de personajes de "Rick and Morty".
- Juego de puzzle interactivo.
- Seguimiento de personajes completados.
- Interfaz amigable y dinámica.

## Estructura del Proyecto

```
index.html
assets/
    icons/
        generate-icons.js
css/
    styles.css
js/
    app.js
```

## Instalación

1. Clona este repositorio en tu máquina local:
   ```bash
   git clone <URL_DEL_REPOSITORIO>
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd puzzle
   ```
3. Abre el archivo `index.html` en tu navegador para comenzar a jugar.

## Uso

1. En la pantalla de inicio, haz clic en "Comenzar" para seleccionar un personaje.
2. Completa el puzzle del personaje seleccionado.
3. Consulta los personajes completados desde la opción correspondiente.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
